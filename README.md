# DRUNKABLE


Party app per raccogliere dati delle serate, generare pagellini e statistiche.


## Setup
1. Installa Expo: `npm install -g expo-cli`
2. Vai in `frontend/` e fai `npm install`
3. Copia le credenziali Firebase in `firebaseConfig.js` (usa `firebaseConfig.example.js` come template)
4. Avvia: `npm start`


## Backend
1. Vai in `backend/functions/` e fai `npm install`
2. Deploy: `firebase deploy --only functions`
